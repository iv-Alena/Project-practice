# **Проектная практика**
1. Бизнес-постановка задачи.
Экосистемы требуют постоянного мониторинга, однако ручное отслеживание видов птиц занимает много времени и зависит от человеческого фактора.
Цель проекта — показать, как можно использовать предобученные модели компьютерного зрения для автоматического обнаружения и классификации птиц по фотографиям. Это поможет экологам, исследователям и волонтёрам оперативно отслеживать биоразнообразие и фиксировать изменения в популяциях птиц
